# 게시물 무한 스크롤링

인스타그램, 페이스북 등 SNS 서비스에서 많이 사용하는 무한 스크롤링 기능을 구현

### 목표

1. 무한 스크롤링
2. 검색어 입력 시 필터링 된 게시물만 보이게 하기
3. 두 개의 탭으로 나누기

### 결과
- [x] 게시물 무한스크롤링으로 불러오기
- [x] 검색바 UI 구현
- [x] 게시물 리스트 구현
- [x] 게시물 상세 페이지
- [x] 검색어를 입력할 시, 150ms Debounce 적용
- [ ] 탭 UI 기능 구현
- [ ] 상세페이지에서 뒤로가기 버튼을 눌러도 기존 내용 불러오기

### 기술 스택
react, react-router-dom, axios, tailwind css
